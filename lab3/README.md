Запуск локально через Makefile:
1. Перейти до папки проєкту
2. Скомпілювати програму, команда: make
3. Запустити виконуваний файл, команда: voting.exe або ./voting (Linux)
4. Щоб очистити об’єктні файли та виконуваний файл, команда: make clean

Запуск через Docker:
1. Перейти до папки проєкту
2. Побудувати docker образ, команда: docker-compose build
3. Запустити контейнер, команда: docker-compose up
4. Зупинити контейнер, команда: docker-compose down
